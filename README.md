# 영어 단어 앱

## 프로젝트 설명
English World는 다양한 카테고리별 영어 단어를 학습할 수 있는 Flutter 애플리케이션입니다.  
이 앱은 여행, 식당, 일상생활, 자동차, 학교, 가족 등 여러 테마의 배너와 함께 사용자의 언어 학습을 돕기 위해 설계되었습니다.
